# Setup

Assuming you created a folder for your project, `cd` to the folder and copy the `Dockerfile` and `requirements.txt` file in there.

Then:

```
sudo docker build -t mydjangoapp .
```

Run it:

```
sudo docker build -t mydjangoapp .
```

Open it:

```
http://localhost:8000/
```
